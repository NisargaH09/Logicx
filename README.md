# AI Voice Based Stroke Risk Detection

## Problem Statement
Stroke can affect a person's speech and voice clarity. This project aims to detect possible stroke risk by analyzing a person's voice using Artificial Intelligence.

## Objective
The objective of this project is to analyze voice patterns and detect early signs of stroke risk using AI technology.

## Technologies Used
- Artificial Intelligence
- Machine Learning
- Speech Processing
- Python

## System Process
1. User records voice through microphone
2. Voice data is collected by the system
3. Audio preprocessing is performed
4. Important voice features are extracted
5. AI model analyzes the voice data
6. System predicts stroke risk level
7. Result is displayed to the user
